# Médias 3D du projet de siège

Déposez ici les rendus/vidéos du bâtiment pour alimenter automatiquement la section "Projet de siège (3D)" de la page À propos.

Formats acceptés:
- Images: .jpg, .jpeg, .png, .webp
- Vidéos: .mp4, .webm, .ogg, .mov

Conseils:
- Préférez des images en 1920px à 3200px de large pour une bonne netteté sur grand écran.
- Nommez les fichiers par ordre (ex: render-01.jpg, render-02.jpg, ...) si vous voulez un ordre spécifique.

Les fichiers seront détectés automatiquement au chargement de la page.
